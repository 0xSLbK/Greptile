# Tetris

A classic Tetris-style puzzle game with a reptilian twist! 


## How to Play
- Use arrow keys (←→) to move pieces
- Use up arrow (↑) to rotate pieces  
- Use down arrow (↓) for soft drop
- Press Space for hard drop
- Press P to pause
